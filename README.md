#Hotel Search
small react app allow you to search in list of hotels to see the available hotels with data range

##Libraries
react
react-router-dom
redux
react-redux
antd
moment
jest
enzyme

##Development
```
git clone https://github.com/Mustafa-AbdAlmogoud/hotels-search.git
cd hotels-search
cp .env-example .env
yarn start
```

##Production
```
git clone https://github.com/Mustafa-AbdAlmogoud/hotels-search.git
cd hotels-search
yarn run deploy
```
